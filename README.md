# Stegbar Legacy CMS to Shopify Migration 

## Project Overview
Stegbar is an international brand with a large-scale website containing both ecommerce and non-ecommerce content.  
The original platform was built on a legacy CMS (Optimizely / Episerver) with a highly complex backend and performance limitations.

Due to frequent server slowdowns and maintenance challenges, the client decided to migrate the entire platform to Shopify while preserving the existing design and user experience.

## Key Responsibilities
- Analyzed and understood the legacy Optimizely/Episerver architecture
- Planned and executed the migration strategy to Shopify
- Recreated legacy page layouts on Shopify with **pixel-accurate design matching**
- Handled large-scale content and product migration planning
- Ensured both ecommerce and non-ecommerce pages worked seamlessly
- Optimized front-end performance compared to the legacy platform

## Challenges
- Extremely complex and non-standard legacy backend
- Very large content and product volume
- Almost every page had a **different custom layout**
- Client requirement to keep the new Shopify site **identical to the legacy design**
- Performance issues on the legacy server affecting development and testing

## Outcome
- Successfully migrated the platform to Shopify
- Preserved all custom layouts and visual consistency
- Improved reliability and performance
- Delivered a scalable and maintainable solution for future growth
